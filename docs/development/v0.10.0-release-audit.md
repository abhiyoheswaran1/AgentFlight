# AgentFlight v0.10.0 Release Audit

Audit date: 2026-06-22

Package version before bump: `0.9.0`

Intended release version: `0.10.0`

## Implementation Summary

AgentFlight v0.10.0 turns the Review Contract into an actionable local review
path:

- Review Contract claims now include source-free proof references for changed
  files, proof statuses, proof gaps, readiness reasons, and suggested proof
  commands.
- Status, Markdown reports, HTML replays, resume prompts, and handoffs now show
  a compact Review Contract review path with a summary, next action, and
  highest-friction claims to inspect first.
- HTML replays now include stable anchors for Review Contract claims,
  review-focus rows, and proof gaps so reviewers can jump from a claim to the
  related local proof area.
- README examples now show the local Review Contract path and an ASCII
  local-review-flow diagram.

The release remains local-first and source-free. It does not add cloud sync,
telemetry, source upload, PR comments, JSON/CI expansion, named verification
profiles, export modes, login, billing, hosted workflows, or GitHub App
behavior.

## Scope Assessment

This is a minor release because it materially improves the review workflow and
rendered review artifacts. The implementation is still focused on one product
spine: actionable, traceable Review Contract review.

No dependency changes were made.
