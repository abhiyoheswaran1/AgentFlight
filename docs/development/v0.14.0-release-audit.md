# AgentFlight v0.14.0 Release Audit

Date: 2026-06-27

## Scope

Release AgentFlight Baseframe Suite Integration v1.

## Included Changes

- Baseframe ProjScan assessment and AgentLoopKit task-contract JSON loading.
- Baseframe session integration context persistence.
- Scope drift detection against allowed and excluded task scope.
- Required verification gate reconciliation against AgentFlight verification
  runs.
- Unified AgentFlight result generation at
  `.baseframe/evidence/<task-id>/agentflight-result.json`.
- `.baseframe/agent-workflow.json` agentflight-section update.
- `agentflight finalize`.
- Status, report, replay, and resume Baseframe sections.
- Public package types and functions for Baseframe contracts/results.
- Documentation and contract fixtures.

## Risk Notes

- Public CLI/API changed with new commands, options, exported types, and exported
  helpers.
- The integration writes local `.baseframe/` evidence but does not add cloud,
  login, billing, GitHub App, or team behavior.
- ProjScan preflight may require human review if scale/complexity risk is
  flagged for the release diff.

## Rollback

If the tag is pushed but npm publish fails, fix forward and retag only if the
tag has not produced a published npm version. If npm publish succeeds with a
critical issue, ship a patch release.
