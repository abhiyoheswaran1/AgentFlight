## Product Summary

AgentFlight v0.16.0 adds a live trust monitor before the final Review Passport.
Users can run:

```bash
agentflight guard
agentflight guard --once
agentflight guard --format json --once
```

Guard reads the same local evidence as `agentflight status`, reports readiness,
changed files, verification counts, proof gaps, Baseframe gates, scope drift,
finish targets, and one next action, and leaves verification execution explicit
through `agentflight verify`.

## Local-First Boundary

This release does not add hosted review, source upload, telemetry, login,
billing, GitHub App behavior, team workflows, or automatic PR comments.

Guard does not run verification commands automatically. It reads local
AgentFlight and Baseframe evidence and renders paths, counts, statuses,
commands, summaries, and artifact targets.
