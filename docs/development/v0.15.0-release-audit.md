# AgentFlight v0.15.0 Release Audit

Date: 2026-06-27

## Scope

v0.15.0 adds `agentflight finish`, Review Passport JSON and Markdown
artifacts, package exports for Review Passport creation, and Baseframe-aware
finish behavior.

## Manual Bug Pass

- Standalone CLI flow from `dist/cli.js`: `init`, `start --task`, edit a file,
  `finish`. Result: exit `0`, Review Passport JSON/Markdown written, handoff,
  report, replay, and resume paths printed, next action was `Share the Review
Passport and handoff packet for scoped review.`
- Baseframe CLI flow from `dist/cli.js`: start from AgentLoopKit task fixture,
  run one passing gate, run one failing gate, leave one missing gate, edit
  allowed/outside/excluded paths, then `finish`. Result: exit `1`,
  `agentflight-result.json` written, gates reconciled as passed/failed/missing,
  scope drift contained only user changes, and Baseframe artifacts stayed in the
  result artifact list.
