# AgentFlight v0.5.1 Release Audit

Audit date: 2026-06-17

Package version before bump: `0.5.0`
Intended release version: `0.5.1`

## Implementation Summary

AgentFlight v0.5.1 is a focused patch for v0.5.0 dogfood findings:

- Terminal `agentflight verify` now prints the stored stderr-preferred
  `outputExcerpt`, matching report and replay behavior.
- Raw stdout and stderr evidence files remain preserved.
- Shared output display helpers compact long suggested proof commands and
  summarize tool-report diagnostics.
- Status, Markdown report, HTML replay, and resume surfaces trim long suggested
  proof commands in dense review areas.
- Markdown report Tooling output keeps AgentLoopKit availability/failure signal
  while reducing repeated doctor diagnostics.
- `.projscan-memory/**` remains suggestion-only guidance for
  `changedFileFilters.ignore`, not a built-in ignored path.
