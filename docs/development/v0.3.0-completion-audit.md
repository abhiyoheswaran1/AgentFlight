## Security And Privacy

Result: pass.

- No telemetry was added.
- No cloud upload was added.
- No login, billing, database, GitHub App, or hosted feature was added.
- Source does not use `fetch`, `axios`, or HTTP upload paths.
- Network-capable behavior is limited to intentional ProjScan/AgentLoopKit `npx` fallback behavior and npm tooling commands.

- Evidence files store command output locally under `.agentflight/evidence/`.
- `.gitignore` excludes `.agentflight/sessions/`, `.agentflight/reports/`, `.agentflight/current/`, and `.agentflight/evidence/`.
