# Roadmap

Current working direction: keep AgentFlight focused on becoming the best
local-first review layer for AI coding sessions. See
[docs/development/product-direction.md](../development/product-direction.md)
for the active post-v0.6.0 priority order.

## MVP

- Local CLI package.
- Safe `.agentflight/` initialization.
- Session start and current handoff.
- Changed-file risk summary.
- Markdown proof report.
- Self-contained HTML replay.
- Resume prompt.
- Doctor checks.
- Defensive ProjScan and AgentLoopKit adapters.
- Verification evidence capture.
- Session events and snapshots.
- Review focus ranking.
- Proof gap detection.
- Review readiness recommendations.
- Config-driven generated/internal changed-file filters.

## Future Free/Open-Core

- Basic local session history.
- Continued local handoff ergonomics.
- First-run workspace hygiene polish.
- Replay navigation and review ergonomics.
- Proof guidance quality improvements.
- Deeper ProjScan-enriched review hints once stable structured output exists.
- More precise git diff stats without including full code by default.
- Structured local automation surfaces where they make review artifacts easier
  to inspect.

## Future Pro

- Searchable local session history.
- Advanced replay exports.
- Multi-repo workspace.
- Custom report themes.
- Advanced policy packs.
- Agent mistake analytics.

## Future Team

- GitHub PR comments.
- Shared policies.
- Team dashboard.
- Audit trail.
- Required verification gates.
- Slack, Linear, and Jira integrations.

## Future Enterprise

- Self-hosted deployment.
- SSO.
- Compliance exports.
- Custom risk rules.
- Support.
