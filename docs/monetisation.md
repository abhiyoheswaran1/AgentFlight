# Monetisation

AgentFlight should be open-core.

## Free

- Local CLI.
- Basic session recording.
- Basic risk report.
- Basic Markdown proof report.
- Basic HTML replay.
- Resume prompt.
- ProjScan and AgentLoopKit integration.

## Future Pro

- Searchable local session history.
- Advanced replay exports.
- Multi-repo workspace.
- Custom report themes.
- Advanced policy packs.
- Agent mistake analytics.

## Future Team

- GitHub PR comments.
- Shared policies.
- Team dashboard.
- Audit trail.
- Required verification gates.
- Slack, Linear, and Jira integrations.

## Future Enterprise

- Self-hosted.
- SSO.
- Compliance exports.
- Custom risk rules.
- Support.

## MVP Boundary

Do not implement billing, Stripe, login, account creation, cloud sync, paid gates, or GitHub App behavior in the MVP. Placeholder commands may state that AgentFlight Pro/Team is not available yet.
